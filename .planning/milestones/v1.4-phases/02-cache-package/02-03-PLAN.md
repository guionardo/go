**cache/memcache/memcache.go** — Memcache Cache[K,V] implementation:
- Package: `memcache`
- Import: `context`, `encoding/json`, `fmt`, `time`, `"github.com/bradfitz/gomemcache/memcache"`, `"github.com/guionardo/go/cache"`
- Struct `Cache[K comparable, V any]`:
  - `client *memcache.Client`
  - `defaultTTL time.Duration`
- Constructor `func New[K comparable, V any](opts ...Option) *Cache[K, V]`:
  - `cfg := defaultConfig(); for _, opt := range opts { opt(cfg) }`
  - `mc := memcache.New(cfg.Servers...)`

  - `mc.MaxIdleConns = cfg.MaxIdleConns`
  - Return `&Cache[K, V]{client: mc, defaultTTL: cfg.DefaultTTL}`
- Helper struct for context cancellation wrapper:
  - ```go
    type memcacheResult struct {
        item *memcache.Item
        err  error
    }
    ```
- Methods:
  - `Get(ctx context.Context, key K) (V, error)`:
    - gomemcache does NOT support context.Context per RESEARCH.md Pitfall 1 (lines 344-361)
    - Wrap the call in a goroutine with select on ctx.Done():
      ```go
      keyStr := fmt.Sprint(key)
      ch := make(chan memcacheResult, 1)
      go func() {
          item, err := c.client.Get(keyStr)
          ch <- memcacheResult{item, err}
      }()
      select {
      case <-ctx.Done():
          var zero V
          return zero, fmt.Errorf("cache/memcache: %w", ctx.Err())
      case r := <-ch:
          if r.err == memcache.ErrCacheMiss {
              var zero V
              return zero, fmt.Errorf("cache/memcache: %w", cache.ErrMiss)
          }
          if r.err != nil {
              var zero V
              return zero, fmt.Errorf("cache/memcache: %w", r.err)
          }
          var value V
          if err := json.Unmarshal(r.item.Value, &value); err != nil {
              var zero V
              return zero, fmt.Errorf("cache/memcache: %w", err)
          }
          return value, nil
      }
      ```
  - `Set(ctx context.Context, key K, value V, ttl ...time.Duration) error`:
    - `data, err := json.Marshal(value)` — if error: `fmt.Errorf("cache/memcache: %w", err)`
    - `expiration := c.resolveTTL(ttl...)` — returns int32 (seconds) for memcache
    - Build `&memcache.Item{Key: fmt.Sprint(key), Value: data, Expiration: expiration}`
    - Wrap `c.client.Set(item)` in goroutine with ctx.Done() select (same pattern as Get)
    - If error: `fmt.Errorf("cache/memcache: %w", err)`
    - Return nil
  - `Delete(ctx context.Context, key K) error`:
    - Wrap `c.client.Delete(fmt.Sprint(key))` in goroutine with ctx.Done() select
    - If error (ignore memcache.ErrCacheMiss on delete — deleting a missing key is not an error in our abstraction): check if `err == memcache.ErrCacheMiss` → return nil (idempotent delete)
    - Otherwise: `fmt.Errorf("cache/memcache: %w", err)`
    - Return nil
  - `GetOrSet(ctx context.Context, key K, setter func() (V, error), ttl ...time.Duration) (V, error)`: Same pattern as other providers
  - `Close() error`: memcache.Client has no Close method — return nil (no-op)
  - `resolveTTL(ttl ...time.Duration) int32`:
    - If `len(ttl) > 0 && ttl[0] > 0`: return `int32(ttl[0].Seconds())`
    - If `c.defaultTTL > 0`: return `int32(c.defaultTTL.Seconds())`
    - Return `0` (no expiry — memcache 0 means no expiry per its protocol)
- All errors wrapped with `"cache/memcache: %w"` per D-12
- Key conversion: `fmt.Sprint(key)` for all external calls

- Methods on `Cache[K, V]`:
  - `func (c *Cache[K, V]) sweepLoop()`:

**cache/postgres/postgres.go** — Postgres Cache[K,V] implementation:
- Package: `postgres`
- Import: `context`, `encoding/json`, `fmt`, `time`, `"github.com/jackc/pgx/v5/pgxpool"`, `"github.com/guionardo/go/cache"`
- Struct `Cache[K comparable, V any]`:

  - `defaultTTL time.Duration`
  - `sweepInterval time.Duration`
  - `stop chan struct{}`
- Constructor `func New[K comparable, V any](opts ...Option) (*Cache[K, V], error)`:
  - NOTE: This constructor returns `(*Cache[K,V], error)` — not just `*Cache[K,V]` — because pgxpool.New can fail (connection string validation, auth failure)
  - `cfg := defaultConfig(); for _, opt := range opts { opt(cfg) }`
  - `pool, err := pgxpool.New(context.Background(), cfg.ConnString)` — if err: return nil, `fmt.Errorf("cache/postgres: %w", err)`
  - Run `CreateTableSQL` on pool: `_, err = pool.Exec(context.Background(), CreateTableSQL)` — if err: return nil, `fmt.Errorf("cache/postgres: %w", err)`
  - Return `&Cache[K, V]{pool: pool, tableName: cfg.TableName, defaultTTL: cfg.DefaultTTL, sweepInterval: cfg.SweepInterval, stop: make(chan struct{})}`, nil
  - Start sweep loop: `go c.sweepLoop()`
  - If the caller passes `WithTableName(...)`, use that table name in all SQL queries via `fmt.Sprintf` interpolation — NOTE: this is NOT injection-safe. Document that table name must be a trusted identifier (not user input). Alternative: use `pgx.Identifier{schema, table}.Sanitize()` for safe quoting.
  - Use `pgx.Identifier{c.tableName}.Sanitize()` for safe SQL identifier quoting.
- Methods:
  - `Get(ctx context.Context, key K) (V, error)`:
    - `query := fmt.Sprintf("SELECT value FROM %s WHERE cache_key = $1 AND (expires_at IS NULL OR expires_at > NOW())", pgx.Identifier{c.tableName}.Sanitize())`
    - `var valueJSON string`
    - `err := c.pool.QueryRow(ctx, query, fmt.Sprint(key)).Scan(&valueJSON)`
    - If `err == pgx.ErrNoRows`: return zero + `fmt.Errorf("cache/postgres: %w", cache.ErrMiss)`
    - If `err != nil`: return zero + `fmt.Errorf("cache/postgres: %w", err)`
    - Parse JSON: `var value V; json.Unmarshal([]byte(valueJSON), &value)` — if error: `fmt.Errorf("cache/postgres: %w", err)`
    - Return `value, nil`
  - `Set(ctx context.Context, key K, value V, ttl ...time.Duration) error`:
    - `data, err := json.Marshal(value)` — if error: `fmt.Errorf("cache/postgres: %w", err)`
    - `expiresAt := c.resolveTTL(ttl...)` — returns `*time.Time` (nil for no expiry)
    - Build UPSERT query:
      ```go
      query := fmt.Sprintf(
          "INSERT INTO %s (cache_key, value, expires_at) VALUES ($1, $2, $3) ON CONFLICT (cache_key) DO UPDATE SET value = $2, expires_at = $3",
          pgx.Identifier{c.tableName}.Sanitize(),
      )
      ```
    - Execute: `_, err = c.pool.Exec(ctx, query, fmt.Sprint(key), string(data), expiresAt)`
    - If error: `fmt.Errorf("cache/postgres: %w", err)`
    - Return nil
  - `Delete(ctx context.Context, key K) error`:
    - `query := fmt.Sprintf("DELETE FROM %s WHERE cache_key = $1", pgx.Identifier{c.tableName}.Sanitize())`
    - `_, err := c.pool.Exec(ctx, query, fmt.Sprint(key))`
    - If error: `fmt.Errorf("cache/postgres: %w", err)`
    - Return nil
  - `GetOrSet(ctx context.Context, key K, setter func() (V, error), ttl ...time.Duration) (V, error)`: Same pattern as other providers
  - `Close() error`:
    - `close(c.stop)` — stop sweep goroutine
    - `c.pool.Close()` — pgxpool.Close is safe to call multiple times
    - Return nil
  - `resolveTTL(ttl ...time.Duration) *time.Time`:
    - If `len(ttl) > 0 && ttl[0] > 0`: return `timePtr(time.Now().Add(ttl[0]))`
    - If `c.defaultTTL > 0`: return `timePtr(time.Now().Add(c.defaultTTL))`
    - Return nil (no expiry)
    - Helper: `func timePtr(t time.Time) *time.Time { return &t }`

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Consumer → Memcache/Postgres provider | Caller's code passes typed K,V parameters |
| Memcache provider → External server | TCP connection to memcached server — network boundary |
| Postgres provider → External database | TCP connection to PostgreSQL — network boundary |
| JSON serialization/deserialization | Value serialization via encoding/json |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-07 | Tampering | SQL injection via table name | medium | mitigate | Use `pgx.Identifier{c.tableName}.Sanitize()` for SQL identifier quoting — prevents SQL injection from table name options. The table name is set at construction, not from user input, providing defense-in-depth. |
| T-02-08 | Information Disclosure | Connection string secrets (Postgres) | high | mitigate | Connection string accepted via functional option `WithConnString(string)` — recommend callers source from `DATABASE_URL` environment variable, not hardcoded values. pgxpool connection string can include password; callers responsible for secret management. |
| T-02-09 | Tampering | JSON deserialization | low | accept | encoding/json is safe against arbitrary data; generic type safety from V parameter. |
| T-02-10 | Spoofing | Memcache server identity | medium | accept | Servers configured via WithServers option; no TLS support in gomemcache (library limitation). Caller should use localhost or trusted network. |
| T-02-11 | Information Disclosure | Postgres sweep best-effort logging | low | accept | Sweep errors logged at Warn level via slog; no sensitive data in logs. |
</threat_model>
