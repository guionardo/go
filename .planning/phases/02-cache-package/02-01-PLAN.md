**cache/mem/mem.go** — In-memory Cache[K,V] implementation following config/provider.go RWMutex pattern:
- Package: `mem`
- Import: `context`, `fmt`, `sync`, `time`, `"github.com/guionardo/go/cache"`
- Struct `Cache[K comparable, V any]`:
  - `mu sync.RWMutex` — per D-05, RLock for reads, Lock for writes
  - `entries map[K]*entry[V]`
  - `defaultTTL time.Duration`
  - `stop chan struct{}` — for sweep goroutine shutdown
- Constructor `func New[K comparable, V any](opts ...cache.Option) *Cache[K, V]`:
  - Creates map, stop channel
  - Applies functional options via interface assertion `opt.apply(&cfg)` where `cfg` is a local `cacheConfig` struct with `defaultTTL time.Duration`
  - Default TTL fallback: if `cfg.defaultTTL == 0` after options, set `cfg.defaultTTL = 5 * time.Minute` (note: still allows no-expiry via explicit `WithDefaultTTL(0)` — handled in resolveTTL)
  - Starts sweep goroutine: `go c.sweepLoop()`
  - Returns `*Cache[K, V]`
- Methods:
  - `Get(ctx context.Context, key K) (V, error)`:
    - RLock, lookup map, RUnlock
    - If not found: return zero + `fmt.Errorf("cache/mem: %w", cache.ErrMiss)`
    - Passive TTL check: if `e.expiresAt != nil && time.Now().After(*e.expiresAt)`:
      - Lock, delete key, Unlock
      - Return zero + `fmt.Errorf("cache/mem: %w", cache.ErrMiss)`
    - Return `e.value, nil`
  - `Set(ctx context.Context, key K, value V, ttl ...time.Duration) error`:
    - Call `c.resolveTTL(ttl...)` for expiration
    - Lock, set entry, Unlock
    - Return nil
  - `Delete(ctx context.Context, key K) error`:
    - Lock, delete from map, Unlock
    - Return nil
  - `GetOrSet(ctx context.Context, key K, setter func() (V, error), ttl ...time.Duration) (V, error)`:
    - First try Get; if no error, return value
    - If error (miss/expired): call setter
    - If setter returns error: return zero + wrapped error
    - Call Set with computed value and ttl
    - Return value, nil
  - `Close() error`: close(c.stop), return nil
  - `resolveTTL(ttl ...time.Duration) *time.Time`:
    - If len(ttl) > 0 && ttl[0] > 0: return `time.Now().Add(ttl[0])`
    - If `c.defaultTTL > 0`: return `time.Now().Add(c.defaultTTL)`
    - Return nil (no expiry)
- Per D-12: all errors wrapped with `"cache/mem: %w"` prefix

**cache/mem/sweeper.go** — Background TTL sweep per D-06:
- Same package `mem`
- `func (c *Cache[K, V]) sweepLoop()`:
  - `ticker := time.NewTicker(1 * time.Minute)`
  - `defer ticker.Stop()`
  - `select` loop: `case <-ticker.C: c.sweep()` or `case <-c.stop: return`
